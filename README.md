# Tempth: A Dimensional Model of Consciousness

**Jenna Zinn | March 2026 | Working Paper**

---

## What Is Tempth?

Tempth (φ) is a proposed second axis of conscious awareness — distinct from physical chronological time — along which the subjective "now" actually moves.

Every major consciousness framework (IIT, GWT, Predictive Processing, Free Energy Principle) models awareness as a function of physical time alone. This paper argues that assumption — dA/dφ = 0 — is the shared structural gap that causes all of them to fail on the same phenomena: dreams, dissociation, trauma, déjà vu, psychedelic states, NDEs, and flow.

Tempth formalizes the missing axis.

---

## Core Claim

> *"The 'me' is not the content of consciousness. It is the organizing coordinate — the dimension from which all conscious states are rendered."*

The ego is not consciousness. It is a projection operator that collapses 2D awareness onto a 1D narrative timeline:

**Ego: A(t, φ) → A(t)***

It ignores the φ axis entirely. This is why it can't integrate dream logic, trauma intrusion, or intuition — they're signal from the second dimension, not noise.

---

## The Formal Framework

Awareness at any moment is defined as:

**A(t, φ)**

Where:
- **t** = physical chronological time
- **φ** = tempth coordinate (φ ∈ ℝ)

A life is not a line through time. It is a path:

**Γ = { A(t, φ(t)) }**

Different states of consciousness have characteristic φ(t) profiles — waking stabilizes φ, dreams let it fluctuate freely, panic compresses it, flow expands it coherently, dissociation decouples it from t entirely.

## The Six-Layer Architecture

1. **The Substrate** — Reality independent of any observer. Static 4D block.
2. **The Renderer** — Moment-to-moment awareness. The last thing to fail clinically. H.M. had this. Clive Wearing has this.
3. **The Ego-UI** — The projection operator. Not consciousness — the interface consciousness uses. Dissolves in dreams, psychedelics, deep meditation, NDEs. Experience does not disappear when it does.
4. **The Frequency Gate** — G(f): A(t,φ) → A_f(t,φ). Controls how much of the φ dimension is accessible at any moment.
5. **The Memory Pipeline** — Two sub-layers: Live Buffer (seconds-minutes continuity) and Persistence Encoder (hippocampal long-term storage). Fully dissociable — proven by H.M.
6. **Tempth (φ)** — The missing axis. Temporal thickness. The dimension awareness actually moves through.

## What It Resolves

- **Weyl's Problem** — If the block universe simply "is," what moves? The φ axis moves. The universe doesn't flow. The gaze does.
- **The Moving Spotlight's infinite regress** — No super-time required. The spotlight is a cognitive architecture operating on φ.
- **Ego dissolution paradox** — Awareness intensifies when the ego collapses because the projection operator's suppression of φ lifts.
- **Trauma intrusion** — Not stuck time. A fixed φ-coordinate the ego cannot suppress.
- **NDE phenomenology** — φ(t) approaching the boundary of what the ego-UI can render.
